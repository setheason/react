HTTTTow to Install and Use VS Code's live Sever Extension

Install | Use

Installation Instructions

  1. In Visual Studio Code , click the extensions icon in the left column tool bar
  2. In the search bar that appears, type live Server
  3. Select the Live Server Extension. It should be the top choice
  4. IN the page for Live Server that appears, click the green install button

Usage Instructions

  • Visual Studio Code must be open to a folder
  • In VS Codes Explorers, right clock (Windows) or Control + Click (macOS) on an HTML file. from the context menu that appears, select Open with Live Server
